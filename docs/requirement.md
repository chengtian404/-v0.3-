# 需求文档 (requirement.md)

## 项目名称

瞭望与问数系统 (DataFinderAgentOS) v0.1

## 项目概述

基于 Tornado 异步 Web 框架构建的轻量级数据查询与分析平台。提供前台用户门户与后台管理系统的双入口架构。

## 功能需求

### 前台用户系统

| 功能 | 说明 |
|------|------|
| 用户注册 | 新用户通过 /regist 注册账号，用户名>=3 位，密码>=6 位 |
| 用户登录 | 密码 PBKDF2-SHA256 安全存储，前端 SHA256 预处理 |
| 用户首页 | 显示欢迎信息、平台数据统计、个人信息卡片 |
| 个人中心 | 查看账号基本信息、修改密码 |
| 退出登录 | 清除 Cookie 并返回登录页 |

### 后台管理系统

| 功能 | 说明 |
|------|------|
| 后台登录 | 独立登录入口 /admin/login |
| 仪表盘 | 统计卡片 + 系统动态 + 系统信息面板 |
| 用户管理 | 用户列表展示、启用/禁用切换、删除用户 |
| 退出登录 | 清除 Cookie 并返回后台登录页 |

## 技术栈

- **后端**: Python 3.11, Tornado 6.x
- **数据库**: SQLite3 (WAL 模式, 零外部依赖)
- **前端**: Layui 2.9.x, 原生 HTML/CSS/JS
- **安全**: XSRF 防护, PBKDF2-SHA256 密码哈希, SQL 参数化查询

## 非功能需求

- 零外部数据库依赖，开箱即用
- 启动时自动建表
- 支持前端 SHA256 密码预处理
- 响应式仪表盘布局
