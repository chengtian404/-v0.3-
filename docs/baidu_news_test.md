# 百度新闻规则测试报告

- 测试日期：2026-07-14
- 测试环境：Windows / Python 3 / DataFinderAgentOS 本机服务
- 测试关键词：`人工智能`
- 请求方式：服务端 GET + 数据库中的 RequestHeaders
- 测试 URL：`https://news.baidu.com/ns`

## 最终结论

**规则可用。** 真实联网测试返回并解析出 10 条新闻，字段包含标题、原文 URL、摘要和新闻图片；随后选择其中 1 条成功写入数据仓库，数据库去重逻辑正常。

抽样结果包括：

- `人工智能蓬勃兴起 锻造高质量发展新引擎`
- `新闻联播：从深圳一条街，看中国人工智能蓬勃发展`
- `3000余项展品将亮相2026世界人工智能大会`

## 稳定性说明

测试中曾观察到百度返回 HTTP 200 但正文为“百度安全验证”的情况，说明百度会基于访问频率或网络出口触发反爬策略。系统已实现以下处理：

1. 验证页不会被解析或保存为新闻。
2. 瞭源状态记录为 `blocked` 并显示具体原因。
3. 管理员可在瞭源管理中重新执行测试。
4. 后续可通过降低频率、增加请求间隔或接入合规新闻 API 提升生产稳定性。
