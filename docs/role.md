# 系统角色与背景 (role.md)

> 本文档由人类维护，AI 在编码过程中参考此文档理解系统定位。

## 系统名称

瞭望与问数系统 (DataFinderAgentOS)

## 系统定位

DataFinderAgentOS 是一个轻量级的数据查询与分析平台，旨在为用户提供便捷的数据发现、查询和可视化服务。系统采用前后台分离的双入口架构：
- **前台用户侧**: 面向普通用户，提供数据查询、浏览和个人信息管理功能
- **后台管理侧**: 面向系统管理员，提供用户管理、系统监控和配置管理功能

## 目标用户

| 角色 | 描述 |
|------|------|
| 普通用户 | 注册用户，可登录系统进行数据查询和查看个人信息 |
| 管理员 | 系统管理员，可管理用户、查看系统状态和配置参数 |

## 核心价值

1. **零外部依赖**: 使用 SQLite3 作为数据库，无需额外安装数据库服务
2. **开箱即用**: 启动即自动建表，无需手动初始化
3. **安全可靠**: PBKDF2-SHA256 密码存储、XSRF 防护、SQL 注入防护
4. **轻量高效**: 基于 Tornado 异步框架，单进程即可承载高并发
