# 测试用例文档 (test_case.md)

> 本文档由人类约定，AI 在收到测试指令时适时维护。
> 测试代码统一放在 `test/` 目录下，格式为 `xxx_case.py`，只许新增，不允许删除。

## 测试规范

- **测试框架**: 不使用 pytest/unittest，采用原生 Python 断言 + 打印输出
- **命名格式**: `{模块名}_case.py`
- **执行方式**: `python test/{模块名}_case.py`
- **数据库**: 测试使用与生产相同的 SQLite 数据库，初始化时自动建表
